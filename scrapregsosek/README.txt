Paket kode untuk otomasi upload dokumen dengan robot berbasis framework Selenium Webdriver
------------------------------------------------------------------

Sebelum digunakan :
- Instal Selenium Webdriver (pip install selenium)
- Tambahkan path lokasi chromedriver.exe ke PATH environment variables

Chromedriver dalam folder ini untuk Google Chrome 106.
Jika menggunakan versi <106, silakan download Chromedriver di https://chromedriver.chromium.org/downloads
atau update Google Chrome ke v103.

Sumber driver lain :
geckodriver (Firefox) : https://github.com/mozilla/geckodriver/releases
Safari : https://developer.apple.com/documentation/webkit/testing_with_webdriver_in_safari

scrap : download dari wilkerstat
convert : convert text to column csv
